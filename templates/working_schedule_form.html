Scheduling problems and linear programming

Simple scheduling problem

Workers will be hired to labour for 8 continous hours, however we do not need same amount of employees durings all 8 continous hours. It can be smarter if we divide into 4 hours those 8 hour shifts. Now we can minimize the amount of workers on labour using linear programming. Each possible 8 hours shifts will be included in two 4 hours group. Our only constraints are the number of workers required in each 4 hours groups. Those constraints can be defined below in the given form, and it will be solved using SIMPLEX method. If facing same problem with different shift schedules, you can still use it.

Solving using Linear Programming

Due to the fact that we can set the problem as a system of linear equations and inequalities and that we have a linear function to minimize (or maximize if necessary) we can use linear programming. This is a fundamental optimization technique. The basic method for solving LP problems is SIMPLEX. This simple scheduling problem is solved using scipy library (scientific computing Python), that can also be used for non-linear optimization.
